# Генератор цветовой схемы пользовательского интерфейса

https://milena42.github.io/colorGenerator/wheel

- по цветовому кругу и из картинок
- учитывает требования к светлоте и насыщенности для элементов
- показывает цвета на макетах

Расширение для браузера:

- генерирует пользовательский набор цветовых переменных
- показывает цвета на веб-страницах (если на странице используются CSS-переменные)
